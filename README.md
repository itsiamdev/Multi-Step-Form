# Multi Step Form 📝

A sleek and interactive multi-step form built with HTML, CSS, and JavaScript. This form guides users through a series of steps to collect personal information including name, email, phone, password, date of birth, address, and pin code. Perfect for user registration or data collection processes! 🚀

## Features ✨
- **Step-by-Step Navigation**: Easy navigation between form steps with Previous/Next buttons
- **Input Validation**: Ensures all required fields are filled before proceeding
- **Responsive Design**: Works beautifully on all devices 📱💻
- **Clean UI**: Modern design with smooth transitions and user-friendly interface

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Use 📋
1. Clone the repository
2. Open `index.html` in your browser
3. Fill out the form step by step
4. Submit when complete!

Enjoy using the Multi Step Form! 🎉
